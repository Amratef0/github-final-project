# Simple Interest Calculator

This project contains a simple Bash script that calculates simple interest based on user input.

## Features

* Accepts principal amount from the user
* Accepts rate of interest
* Accepts time period
* Calculates simple interest using the standard formula
* Displays the result in a user-friendly format

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## Files

* `simple-interest.sh` – Bash script for calculating simple interest
* `README.md` – Project documentation
* `LICENSE` – Apache 2.0 License

## How to Run

1. Clone the repository.
2. Navigate to the project directory.
3. Make the script executable:

```bash
chmod +x simple-interest.sh
```

4. Run the script:

```bash
./simple-interest.sh
```
